# ePay's Transfer - Premium Money Transfer Webapp

A comprehensive international money transfer platform built with Next.js, featuring competitive exchange rates, secure transactions, and an innovative business network system.

## 🌟 Features

### Core Functionality
- **Public Exchange Rate Checking** - No registration required
- **Secure Money Transfers** - Cross-border transfers with competitive rates
- **Real-time Rate Updates** - Live exchange rate monitoring
- **Mobile-First Design** - Fully responsive across all devices

### ePay's Business Network
- **6-Level Commission System** - Earn 0.2% to 3% commission
- **Insider Network** - Process transfers and earn rewards
- **Business Dashboard** - Track earnings and performance
- **Referral System** - Build your network and increase earnings

### Security & Compliance
- **USA Licensed** - FinCEN MSB Registration
- **International Certified** - ISO 27001 Compliant
- **AML/KYC Compliant** - Full regulatory compliance
- **Bank-Level Security** - Enterprise-grade encryption

## 🚀 Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS with custom design system
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: NextAuth.js with multiple providers
- **State Management**: React Query for server state
- **UI Components**: Custom component library
- **Icons**: Lucide React
- **Notifications**: React Hot Toast

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd epay-transfer
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env.local
   ```
   Fill in your environment variables in `.env.local`

4. **Set up the database**
   ```bash
   npm run db:generate
   npm run db:push
   ```

5. **Run the development server**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to view the application.
